# HimForbidden

![适用版本](https://img.shields.io/badge/适用版本-1.19+-blue)

此插件的功能是设置违禁词并屏蔽

违禁词需要在config.yml中配置，我已经给出了实例

修改完配置文件config.yml后需要输入/hfreload来重载此插件
